# Вместе — онлайн-кинотеатр

Простой сайт: загружаешь видео, скидываешь ссылку на комнату другу — смотрите синхронно, с чатом.

## Что внутри
- `server.js` — сервер (Node.js + Express + Socket.io + Multer)
- `public/index.html` — весь фронтенд (вход в комнату, плеер, чат)
- `uploads/` — сюда будут падать загруженные видео

## Как запустить с телефона (без компьютера)

Проще всего через **Railway** — там можно загрузить проект прямо из браузера телефона.

1. Зарегистрируйся на [railway.app](https://railway.app) (можно через GitHub или Google)
2. Создай пустой репозиторий на GitHub через мобильное приложение GitHub или сайт github.com:
   - New repository → назови, например, `watch-together`
   - Загрузи туда все файлы этого проекта (в GitHub есть кнопка "Add file → Upload files", это работает в мобильном браузере)
3. В Railway: New Project → Deploy from GitHub repo → выбери свой репозиторий
4. Railway сам определит Node.js проект и запустит `npm install` + `npm start`
5. В настройках проекта включи **Public Networking**, чтобы получить публичную ссылку (что-то вроде `watch-together-production.up.railway.app`)
6. Готово — открывай ссылку с телефона и зови друзей

**Альтернатива — Render.com**: тот же принцип (New Web Service → подключить GitHub репозиторий → Build command `npm install`, Start command `npm start`), тоже полностью настраивается из браузера.

## Важно про размер видео и хостинг
- Бесплатные тарифы Railway/Render дают ограниченное место на диске (обычно 1–5 ГБ) и он **не постоянный** — при переразвёртывании файлы могут удалиться
- Для фильмов лучше не хранить видео прямо на сервере, а подключить облачное хранилище (например, Backblaze B2 или Cloudflare R2 — у обоих есть бесплatный тариф). Если понадобится — могу переделать код под это
- Учитывай отдачу (bandwidth) — у бесплатных тарифов есть лимиты в месяц

## Как пользоваться
1. Открой ссылку на сайт
2. Введи имя и код комнаты (просто придуманное слово — его же скажи другу)
3. Загрузи видеофайл кнопкой "📁 Загрузить видео" (или выбери уже загруженное из списка)
4. Плеер, пауза и перемотка синхронизируются у всех, кто в комнате
5. Рядом — чат
